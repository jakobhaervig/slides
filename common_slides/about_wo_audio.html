Slide technicalities

  • Navigate slides using arrow keys
    • "←" and "→" move between major sections
    • "↑" and "↓" dig deeper into current topic
    • Press "esc" shows an overview of slides ("esc" to return)
  • Cinema mode
    • Press "f" or top, right corner to go into cinema mode
  • Slide philosophy
    • Resonably standalone. I.e., works for both "live" presentations and "asynchronously" presentations
    • Easily accessible and only relies on a simple web browser
    • You are always browsing the latest version
  • More info
    • Based on the reveal.js framework
    • Source code freely available at https://github.com/jakobhaervig/courses/
    • Download these slides as .pdf file
